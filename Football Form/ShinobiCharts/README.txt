Getting started with ShinobiCharts
=========================================

1. Open the DMG
2. Read and accept the License
3. Run the pkg installer to add the framework and docs to Xcode (optional but recommended) 
4. Drag the ShinobiCharts folder to your desktop to find the samples 
5. Read the user-guide in the docs which has a QuickStart section
6. Try out some of the sample projects
7. Start creating amazing charts!

* FAQ: http://www.shinobicontrols.com/shinobideveloper
* Quick Start Guide: http://www.shinobicontrols.com/shinobicharts/quickstartguide/

** Note: When creating a project you will need to add the following frameworks:
- Security.framework (Trial version only)
- QuartzCore.framework
- OpenGLES.framework
- CoreText.framework (Version 2.3+)
- libstdc++.dylib (Version 2.3+) **

** Note: The framework is still available in the ShinobiCharts folder if you don't wish to install it to Xcode. **
